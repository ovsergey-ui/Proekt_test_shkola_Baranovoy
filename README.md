# Локальный Анализатор Документов (Offline)

Полностью локальное веб-приложение на `index.html` без сервера и без API.

## Что уже установлено

В проект уже скачаны оффлайн-библиотеки:

- `libs/pdf.min.js`
- `libs/pdf.worker.min.js`
- `libs/mammoth.browser.min.js`

## Как запустить

1. Откройте `index.html` в Chrome или Edge (рекомендуется версия 86+).
2. Нажмите **Выбрать папку** и дайте доступ к директории с документами.
3. Нажмите **Анализировать** для построения статистики и поиска шаблонов.
4. Выберите шаблон, заполните JSON в блоке генерации, нажмите **Генерировать**.
5. Укажите имя и место сохранения через системный диалог.

## Проверка оффлайн-режима

1. Отключите интернет.
2. Перезагрузите страницу.
3. Проверьте статус парсеров в интерфейсе:
   - `PDF: OK`
   - `DOCX: OK`

Если оба статуса `OK`, приложение работает полностью оффлайн.

## Ограничения текущей версии

- PDF извлекается как простой текст (без сложной структуры таблиц).
- DOCX обрабатывается через raw text extraction.
- Для больших папок обработка может занять время (это ожидаемо).
